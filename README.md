# Mini Dashboard

## Features
- Mocked login with hardcoded credentials (email: `admin@example.com`, password: `admin123`)
- Dashboard with mock data table
- Filtering & sorting
- Basic routing
- TypeScript support
- Unit test for Table component
- GitHub Actions CI

## Setup
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Test
```bash
npm run test
```
